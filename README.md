# Student Course Registration System

Java-based application to manage student course enrollment.

## Features

- Student login
- Enroll in courses
- Drop courses
- View registered classes
- Database connectivity

## Technologies

- Java
- JDBC
- Object-Oriented Programming

## Files included

- Course.java
- DBUtil.java
- dropClass.java
- Enroll.java
- login.java
- myClassRegistered.java
